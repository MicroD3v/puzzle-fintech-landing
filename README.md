# Puzzle Fintech 🧩

Built at 4AM. Looks like it wasn't.

A fintech SaaS landing page focused on motion, detail,
and UI that actually feels alive.

## Stack

React · Vite · Tailwind CSS · Framer Motion · Recharts

## Features

- Spring physics custom cursor
- Animated liquidity charts
- Bento grid layout
- Monthly/Yearly pricing toggle
- Glass navbar
- Dark mode

## Getting Started

npm install
npm run dev
